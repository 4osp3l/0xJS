// Installation

Up and running
in under 2 minutes.

Requires Python 3.7+ and a free Google Gemini API key. That's it.

1
Clone the Repository
Pull the source code and navigate into the project directory.
git clone https://github.com/4osp3l/0xJS
cd 0xJS
2
Install Dependencies
Install the required Python packages via pip.
pip install -q -U google-genai
pip install python-dotenv
pip install requests
3
Configure API Key
Set your Google Gemini API key using the built-in config command.
python main.py --mode config \
  --key "AIzaSyYourActualAPIKeyHere"
4
Start Scanning
Run the tool against any JavaScript file or URL.
python main.py
one-liner 
git clone https://github.com/4osp3l/0xJS.git && cd 0xJS && python main.py
